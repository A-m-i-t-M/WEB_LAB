Global variables are used to store data which might be required all throughout the program. The scope of local vairables is only within the functions in which they are defined. There are a total of 3 different kinds of functions that we discussed in todays class

  • Function Declaration (normal usage of a function)
  • Function Expression (we assign the function to a vairable over here)
  • Anonymous function (nameless funtion)
Another concept covered in here is that of Hoisting. So basically, whenever a variable (which is not yet declared/initialised) is called or referenced to, by defualt its value is set to "undefiined" unlike the traditional scenario in languages like Java and all where we would get Compilation
Lastly, we also made use of Lexical scoping where basically the nested functions are able to access the vairables that are present in their parent functions (going up in the hierarchy)
